# Story illustrations

Drop the generated Ghibli-style PNGs from `GHIBLI_PROMPTS.md` into this folder.

Expected filenames (case-sensitive):

- `login-first-light.png`
- `dashboard-quiet-classroom.png`
- `chat-listening-tree.png`
- `quiz-seeds-of-curiosity.png`
- `analytics-reading-the-sky.png`
- `homework-path-home.png`
- `reteach-pond-return.png`
- `reports-pressed-pages.png`
- `settings-library-shelf.png`

Until a file is dropped here, the matching screen will show a soft gradient
placeholder labelled with the scene name — UI layout is preserved. Once a file
is added, the next page reload picks it up automatically.

Recommended size: at least 1080×1920 (9:16) for portrait crops, or 1920×1080
(16:9) if you re-prompt the master template for horizontal banners.
